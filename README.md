# Full-stack-Blogging-App
Full-stack Blogging App Devopic Intership CodeAlpha
A production-ready full-stack blogging application built with Spring Boot 3 (Java 17)
and Thymeleaf. Features user authentication with Spring Security (BCrypt), H2 in-memory
database, and JaCoCo test coverage reports. CI/CD pipeline powered by Jenkins with
Maven, artifacts managed via Nexus, Docker image deployed to Kubernetes on AWS EKS
provisioned with Terraform.
